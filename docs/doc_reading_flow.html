02Logic code (mapping theo file)

  1. Nhận & làm sạch input — agent.py:128-132. Guardrail an toàn: strip, cắt độ dài, chặn input rỗng.
  2. Soạn prompt theo định dạng ReAct — agent.py:65-91. Bơm danh sách tool để LLM biết gọi cái gì.
  3. Gọi LLM (Gemini) — agent.py:104-126. Bọc try/except + retry + backoff; lỗi mạng không làm sập loop.
  4. Parse output — agent.py:171-181. Regex tách Final Answer: (xong) hoặc Action: tool(args) (đọc doc).
  5. ⭐ ĐỌC TÀI LIỆU — agent.py:208 → lab_tools.py:63 → loader.py:33-34. _execute_tool() gọi search_lab_docs() → loader.search() → open(embedded_labs.json). Đây chính là chỗ đọc doc.
  6. Đưa nội dung doc lại cho LLM — agent.py:214. Nối Observation: vào transcript rồi lặp để Gemini tổng hợp câu trả lời.
  7. Kết thúc an toàn — agent.py:200-206, 222-232. loop-guard / max_steps / fallback về Observation chính xác cuối cùng.

Phần cốt lõi đọc doc (rút gọn):

# src/knowledge/loader.py — nơi THỰC SỰ mở & đọc tài liệu
def load():
    global _CACHE
    if _CACHE is None:
        with open(_DATA_PATH, encoding="utf-8") as f:   # ← ĐỌC DOC
            _CACHE = json.load(f)               # cache để không đọc lại
    return _CACHE

# src/tools/lab_tools.py — tool mà agent gọi
def search_lab_docs(args):
    hits = loader.search(args, max_results=5)  # tìm trong doc, không phân biệt dấu
    return render(hits)                       # → trả về Observation cho LLM
